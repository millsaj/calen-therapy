Project Overview
----------------

Calen Therapy Website is a professional web application built with Next.js 14, TypeScript, and Tailwind CSS.

It provides information and services for therapy clients, featuring two primary therapists: Helen Mills Porter and Carl Porter.
The site is statically generated and deployed via Netlify.

Sitemap: https://calentherapy.co.uk/sitemap.xml
